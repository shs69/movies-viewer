# Movies Viewer
Приложение для просмотра фильмов, построенное на React + Vite + React Router.

## Установка

```bash
pnpm install
```

## Настройка

1. Скопируйте файл `.env.default` в `.env`:

```bash
cp .env.default .env
```

## Запуск

### Режим разработки

```bash
pnpm dev
```

Приложение будет доступно по адресу `http://localhost:5173`

### Сборка

```bash
pnpm build
```

### Предпросмотр сборки

```bash
pnpm preview
```

### Линтинг

```bash
pnpm lint
```

## Стек технологий

- **React 19** — UI библиотека
- **React Router 7** — маршрутизация
- **Material UI** — компоненты интерфейса
- **Vite** — сборщик
- **TypeScript** — типизация
- **Axios** — HTTP клиент

## Структура проекта

```
src/
├── components/       # Переиспользуемые компоненты
├── routes/           # Компоненты страниц и loader'ы
├── utils/            # Утилиты и хелперы
├── client.ts         # API клиент
└── main.tsx          # Точка входа
```
